# OpsFlow — Product Requirements

## Vision

Build a universal operations platform for service businesses that scales from solo users to teams and adapts across industries.

## Target users

- Barbers, salons, nail techs
- Car wash / detailing businesses
- Freelancers

## Core entities

- Organization (tenant)
- User
- Resource
- Customer
- Service
- Booking
- Workflow

## Core features

- Booking system
- CRM
- Payments tracking
- Operations workflows
- Dashboard analytics
- Customer booking portal
- In-app chat system

## AI features

- Booking assistant
- Setup assistant
- Business insights
- CRM auto-tagging

## Non-functional requirements

- Fast performance (target: typical queries under 200ms)
- Multi-tenant isolation
- Scalable architecture
